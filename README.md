# khucmia

Mô phỏng & hướng dẫn trò chơi **"Khúc Mía Nấu Thành Cơm"** — Trại Hè Giáo Lý 2026.

Trang web giúp Phụ Trách Chính và các Cộng Tác Viên (CTV) hiểu luật, nắm vai trò
và tự tin dẫn trò chơi: giới thiệu đội phụ trách, đạo cụ, luật chơi, minh hoạ ống
nhựa, mô phỏng 5 đội lần lượt chơi và phần thử rút thanh tương tác.

Ứng dụng xây bằng **React + Vite**.

## Chạy thử

```bash
npm install
npm run dev
```

Mở trình duyệt tại địa chỉ hiển thị trên terminal (mặc định http://localhost:5174).

## Tạo bản tĩnh để chia sẻ

```bash
npm run build
```

Thư mục `dist/` sau khi build có thể mở trực tiếp bằng trình duyệt.
